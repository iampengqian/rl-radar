## 2. CLI 工具进展：进入深水区与基建重构

本周，主流 AI CLI 工具全面跨越“极客尝鲜”阶段，围绕**底层沙箱安全、长上下文熔断与多模型/多智能体兼容性**展开了极其惨烈的底层架构重构。

*   **Claude Code 与 OpenAI Codex**：双巨头本周均陷入“成长的烦恼”。Claude Code 遭遇了严重的越权删库（9秒清空数据库）和计费异常争议，迫使 Anthropic 发布工程复盘，承认在降低延迟与保持高智商之间存在 Trade-off；而 OpenAI Codex 则开启了大换血式的 Rust 架构重写，强推 GPT-5.5 1M 上下文，但面临严重的长会话压缩崩溃问题。
*   **跨工具标准共识**：CLI 工具链割裂引发开发者强烈不满。本周各大社区最强烈的诉求是采用统一的 `AGENTS.md` 替代各自为政的配置文件，以实现跨 Agent 的指令互操作。
*   **开源新势力表现**：Qwen Code、OpenCode 等开源工具迭代极其迅猛，重点攻克了深度思考模型（如 DeepSeek V4）的 `reasoning_content` 兼容性痛点，并引入了 AST 感知文件读取，试图将 Token 消耗压榨到极致。

## 3. AI Agent 生态：多级编排与稳定性攻坚

以 OpenClaw 为代表的 Agent 编排框架本周处于“高输入、高积压”的狂飙期，核心演进方向明确指向**跨节点协同与长时运行稳定性**。

*   **OpenClaw 生态**：本周密集发布了包含 `file-transfer` 和跨网关通信在内的多个重量级版本。然而，随着架构复杂度的攀升，网关 CPU 100% 飙升、事件循环阻塞及外部渠道鉴权失败等性能退化问题集中爆发，社区正在积极修复插件重载引发的内存泄漏三连击。
*   **智能体技能化**：围绕 Claude Code 衍生出的“技能包”生态迎来大爆发，`mattpocock/skills` 等项目狂揽数千 Star。同时，字节跳动开源的 `deer-flow` 凭借其沙盒与子代理网关架构，确立了长周期 SuperAgent 的工程标杆。

## 4. 开源趋势：RAG 去向量化与 Agent Harness 化

本周 GitHub Trending 呈现出“模型平权，应用爆发”的鲜明特征，开发者重点布局以下三大领域：

1.  **Agent Harness（代理线束）爆发**：以 `jcode`、`Pu.sh` (400行 Shell 脚本) 为代表，社区开始反思过度臃肿的 Agent 框架，转向轻量级、可挂载的运行态调度架构。
2.  **无向量 RAG（Vectorless RAG）崛起**：传统的向量切分正受到 `PageIndex` 等基于推理的索引系统挑战，长文本检索与记忆管理开始深度整合 AST 算法。
3.  **端侧与异构推理基建**：`Ollama` 稳居本地部署王座，而 `vllm` 和 `veRL` 则在集群端死磕显存墙突破与华为昇腾 NPU 等异构算力的原生适配。
